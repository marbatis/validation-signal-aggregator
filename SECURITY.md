# Security Policy

## Supported Versions

This repository is maintained on the `main` branch only.
Security fixes are applied to `main`.

## Reporting a Vulnerability

Do not open public GitHub issues for security reports.

Use GitHub Private Vulnerability Reporting when available, or email:
`marbatis@hotmail.com`

Please include:
- repository name
- reproduction steps
- impact assessment
- affected files/paths

We will acknowledge receipt and provide next steps.
